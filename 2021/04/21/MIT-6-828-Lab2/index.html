boot_alloc() 函数详解

boot_alloc()函数是为了后面page_alloc函数的初始化而服务的。函数输入为待分配的的字节数。主要作用是在分配的时候按照页面大小（4K）

Part 2: Virtual Memory

logical address -> linear address(virtual) -> physical address

两次地址转换，分别为Segment Translation和Page Translation。结合段页内存管理系统来理解。

Page-Level Protection

https://pdos.csail.mit.edu/6.828/2018/readings/i386/s06_04.htm

虚拟地址，也就是线性地址，被拆成了三部分，都是一种索引index，分别索引的是Page Directory, Page Table, Page Frame。从page directory中读出page table 的地址，在从读到的page table地址中读到page frame的地址，索引page frame之后，就得到相应物理地址上的内容。

对于开发者来说，page directory, page table都是两个数组，拿到page directory的头部指针，和虚拟地址一起，就可以确定物理地址
