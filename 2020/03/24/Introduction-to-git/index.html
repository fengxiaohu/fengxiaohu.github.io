Git 是什么

Git is a free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency

[]: https://git-scm.com/ “Git 官网链接”

​ Git 是一个开源的分布式版本控制系统。[Linus Torvalds](https://baike.baidu.com/item/Linus Torvalds/9336769)（写出Linux的dalao） 为了帮助管理 Linux 内核开发而开发的一个开放源码的版本控制软件。

​ 通常来说在大型的，需要多人协同的一些项目中使用版本控制软件是有必要的。举个例子来说，当你需要反反复复修改一个word文档时候，那么你通常都要标注初稿，终稿等。那么我们可以想象一个需要多人协同的word文档，自然而然对于版本控制的需求就更加迫切，否则就会乱成一锅粥。
